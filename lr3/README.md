# ЛР 3 — LSD Radix Sort (дек) + поиск методом Райта

Цифровая сортировка через **дек на двусвязном списке** и поиск подпоследовательности **методом Райта**.

## Структура данных

Дек (`DNode{value, prev, next}`) — вставка и удаление с обоих концов за O(1).

## Алгоритм сортировки

LSD Radix Sort через 10 деков-корзин: на каждом разряде элементы уходят через `pop_front` и возвращаются через `push_back`. FIFO даёт устойчивость.

Отрицательные числа: делятся на две группы (отрицательные по модулю и положительные), каждая сортируется независимо, затем объединяются.

## Алгоритм поиска (Райт)

Старт с середины дека (`size/2` шагов от головы). Если текущий элемент больше ключа — идём назад (`prev`), меньше — вперёд (`next`). Среднее число просмотров ≈ n/4 против n/2 у линейного поиска.

## Сборка и запуск

```bash
make
./sort input.txt              # ключи поиска вводятся с клавиатуры
./sort input.txt 75 -73 999  # ключи поиска как аргументы
```
