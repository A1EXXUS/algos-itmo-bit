# ЛР 0 — Численные методы решения уравнений

Решение полиномиального уравнения до 6-й степени на Python.

## Алгоритм

Программа последовательно понижает степень уравнения:

1. Находит один корень численным методом
2. Делит полином на `(x − корень)` по схеме Горнера
3. Повторяет до линейного уравнения

Выбор метода зависит от текущей степени:

| Степень | Метод |
|---------|-------|
| 6+ | Комбинированный (Ньютон + хорды) |
| 5 | Метод хорд |
| 4 | Метод касательных (Ньютон) |
| 3 | Формула Кардано |
| 2 | Дискриминант (теорема Виета) |
| 1 | Линейное уравнение |

## Запуск

```bash
python3 main.py
```

Программа запросит коэффициенты `a..n` уравнения:

```
ax⁶ + bx⁵ + cx⁴ + dx³ + kx² + mx + n = 0
```
