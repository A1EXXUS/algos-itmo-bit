# Алгоритмы и структуры данных — ИТМО БИТ

Лабораторные работы по курсу «Алгоритмы и структуры данных», Университет ИТМО, направление БИТ.

## Структура

| Лаба | Тема | Язык |
|------|------|------|
| [lr0](lr0/) | Численные методы: решение уравнений (метод хорд, метод Горна, формула Кардано и др.) | Python |
| [lr1](lr1/) | LSD Radix Sort — три структуры хранения: массив, связный список, вектор | C |
| [lr2](lr2/) | LSD Radix Sort — сортировка через кольцевые очереди | C |
| [lr3](lr3/) | LSD Radix Sort — дек на двусвязном списке + поиск подпоследовательности (алгоритм Райта) | C |
| [lr4](lr4/) | LSD Radix Sort — стек на связном списке + поиск подпоследовательности (алгоритм Бойера–Мура) | C |

## Сборка (lr1–lr4)

```bash
cd lr1   # или lr2, lr3, lr4
make
./sort input.txt
```

## Состав каждой лабы

- `*.c` / `*.h` — исходный код
- `Makefile` — сборка
- `input.txt` — тестовые данные
- `flowcharts/` — блок-схемы алгоритмов
- `report.tex` / `отчет*.pdf` — отчёт
