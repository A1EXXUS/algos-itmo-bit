# ЛР 2 — LSD Radix Sort через кольцевые очереди

Цифровая сортировка (LSD Radix Sort), где роль корзин играют **кольцевые очереди** на базе односвязного списка.

## Алгоритм

На каждом разряде элементы распределяются по 10 очередям (по цифре разряда) через `enqueue`, затем собираются обратно через `dequeue`. FIFO-порядок обеспечивает устойчивость сортировки.

Поддерживаются отрицательные числа: они сортируются отдельно по модулю, затем прикладываются в обратном порядке.

**Сложность:** O(d × (n + 10)) по времени, O(n + 10) по памяти.

## Сборка и запуск

```bash
make
./sort input.txt
```
