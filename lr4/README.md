# ЛР 4 — LSD Radix Sort (стек) + поиск Бойера–Мура + красно-чёрное дерево

Цифровая сортировка через **стек на динамическом массиве**, поиск подпоследовательности **методом Бойера–Мура** и запись результата в **красно-чёрное дерево**.

## Структуры данных

- **Стек** (`data[]`, `top`, `capacity`) — LIFO, ёмкость удваивается при переполнении, амортизированное O(1) на `push`.
- **Красно-чёрное дерево** — для хранения отсортированного результата с визуализацией в терминале.

## Алгоритм сортировки

LSD Radix Sort через 10 стеков-корзин. Двойной реверс при сборке компенсирует LIFO и сохраняет устойчивость. Отрицательные числа обрабатываются отдельно.

После сортировки весь стек вставляется в красно-чёрное дерево, которое выводится в виде ASCII-дерева.

## Алгоритм поиска (Бойер–Мур)

Стек сериализуется в строку `";v1;v2;...;"`, искомая подпоследовательность — в паттерн `";k1;k2;...;"`. Разделители `;` исключают ложные совпадения внутри токена (например, `-1` не найдётся внутри `-100`).

## Сборка и запуск

```bash
make
./sort input.txt                    # ключи поиска вводятся с клавиатуры
./sort input.txt 75 -73 2 999      # ключи поиска как аргументы
```
