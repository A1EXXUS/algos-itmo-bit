# ЛР 1 — LSD Radix Sort (массив / вектор)

Цифровая сортировка (LSD Radix Sort) с тремя структурами хранения на выбор.

## Структуры данных

| Режим | Структура | Описание |
|-------|-----------|----------|
| 1 | Статический массив | Фиксированный буфер `int[1000]` |
| 2 | Динамический массив | `malloc` под точное число элементов |
| 3 | Вектор | Авторасширяемый массив с удвоением ёмкости |

## Алгоритм

LSD Radix Sort: сортировка от младшего разряда к старшему через counting sort.

**Сложность:** O(d × (n + 10)), где `d` — число разрядов максимального элемента, `n` — количество элементов. При фиксированных `d` и `k` — линейная **O(n)**.

## Сборка и запуск

```bash
make
./sort input.txt
```

Программа предложит выбрать структуру (1/2/3) и выведет отсортированный массив.
